# RV32I-Assembler
RISC-V 32-bit Base Integer Instruction Set (RV32I) Assembler

## Usage
```
git clone https://github.com/LostParadise07/RV32I-Assembler.git
cd RV32I-Assembler
g++ -std=c++17 main.cpp -o main.out
./main.out
```
